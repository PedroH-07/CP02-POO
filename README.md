# FiapDelivery

Refatoracao do codigo legado aplicando POO e Clean Code.

## O que foi feito
- Encapsulamento: atributos privados nas classes.
- Heranca: `Caminhao` e `Moto` herdam de `Veiculo`.
- Associacao: `Rota` relaciona `Veiculo` e `Pacote`.
- Clean Code: nomes claros e estrutura organizada em `app/` e `model/`.
